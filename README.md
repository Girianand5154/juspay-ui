# Juspay UI Project

## Setup and Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Girianand5154/juspay-ui
   cd Feature_Giri
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm start
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```
